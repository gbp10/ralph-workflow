# Current Ralph Task

This file is auto-generated by the orchestrator or manually created for single-story execution.

## Usage

### Option 1: Manual Creation
Create this file with your task details, then run:
```bash
/ralph-loop "$(cat .claude/ralph-workflow/prompts/CURRENT_TASK.md)" --completion-promise "DONE"
```

### Option 2: Orchestrator Generation
Run the orchestrator and it will generate this file from your stories JSON:
```bash
.claude/ralph-workflow/scripts/ralph-orchestrator.sh
```

## Template

```markdown
# Current Ralph Task: US-001 - Task Title

## User Story
**As a** [role],
**I want** [goal],
**So that** [benefit].

---

## Task Details

[Your detailed task prompt here]

---

## Acceptance Criteria

- [ ] Criterion 1
- [ ] Criterion 2
- [ ] Criterion 3

---

## Completion Promise

When ALL criteria are met, output:

<promise>US-001 COMPLETE</promise>
```
